# mallocあれこれ
## malocとは？
mallocとはメモリを確保/解放するライブラリ関数のこと。<br>
malloc() -> メモリ確保<br>
free() -> メモリ解放<br>
セットで行う！らしい。<br>

あとは、mallocは大きなメモリしか管理できないOSと小さいサイズのメモリを使いたいapplicationの橋渡し的存在らしい。中間管理職は大変そうですね。<br>
